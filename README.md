# NexusChat

Telegram-inspired real-time chat frontend built with Angular.

## Features

- Telegram-style chat layout
- Responsive mobile chat UI
- Chat sidebar with search
- Conversation switching
- Message bubbles
- Message composer
- Typing indicator
- Online status
- Unread badges
- Message status UI
- Socket-ready service for future backend integration

## Tech Stack

- Angular
- TypeScript
- SCSS
- Signals
- Standalone Components

## Run locally

```bash
npm install
ng serve
